# web_analize

日本株・米国株の個人用市場監視ダッシュボード。Django 5.2 / PostgreSQL 17。

「市場全体をスキャンし、自分が見るべきものだけを浮かび上がらせる」ことを目的にしている。

## 機能

| 機能 | 内容 |
|---|---|
| 時価総額ランキング | 日本株の時価総額を横棒グラフで表示。17業種で絞り込み |
| 出来高急増ランキング | 出来高の統計的異常度（対数z-score）で順位付け。単純な前日比と違い、銘柄ごとのばらつきを考慮する |
| 銘柄別指標 | PER/PBR/ROE/ROA/配当利回り/自己資本比率と、四半期20期分の推移（PERはTTMベース） |
| 銘柄カルテ | IR資料を読みながら全銘柄共通の雛形に記入する詳細分析。中期経営計画の進捗と独自KPIをグラフ化 |
| 売買日記 | 売買判断の記録と後日の振り返り。記録後の値動きと概算損益を自動追跡 |

グラフは Apache ECharts。銘柄別指標ページは全銘柄のデータを一括で埋め込み、
銘柄切替時にサーバーへ再アクセスしない構成。

## データソース

- [J-Quants API](https://jpx-jquants.com/) V2 — 日本株の株価・決算（V1は廃止済み）
- yfinance — 米国株の株価
- NASDAQ Trader — 米国株ティッカー一覧

## セットアップ

環境バージョン・デプロイ手順・既知の注意点は **[CLAUDE.md](CLAUDE.md)** に集約している。

`config.json`（APIキーとDB接続情報）はリポジトリに含まれないため、別途設置が必要。

```bash
python3 -m venv venv
./venv/bin/pip install -r requirements.txt
./venv/bin/python manage.py migrate
```
